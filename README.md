# README

* Ruby version -- 2.5.1

* System dependencies -- Postgresql 10

* Database creation

    1. Install posgresql
    2. Create user `Nanoblogs` with password `Nanoblogs`
    3. Create databases `Nanoblogs_development`, `Nanoblogs` and `Nanoblogs_test`

* Deployment instructions
    -- run `$ rails server`
